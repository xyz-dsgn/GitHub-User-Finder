# GitHub-User-Finder
Программа для поиска пользователей GitHub

## Описание программы 
GUI-приложение для поиска пользователей на GitHub с помощью официального API. Позволяет сохранять понравившихся пользователей в избранное (хранится в `favorites.json`).
## Требования для запуска 
Что нужно установить на компьютер: 
- Python 3.10 или выше 
- Библиотеки: `pip install tkinter`, `pip install requests`
## Как запустить 
```bash 
git clone https://github.com/xyz-dsgn/GitHub-User-Finder
cd GitHub-User-Finder
python main.py
```
## Как пользоваться
1. Запустите программу.
2. Введите в поле поиска логин или имя пользователя GitHub (например, torvalds или mojombo).
3. Нажмите кнопку «Поиск».
4. В списке появятся найденные пользователи.
5. Выберите пользователя и нажмите «Добавить в избранное» — он сохранится в файл favorites.json.
## Пример работы 
1. В поле поиска введите torvalds.
2. Нажмите «Поиск».
3. В списке появится пользователь torvalds (Линус Торвальдс).
4. Выберите его и нажмите «Добавить в избранное».
5. В файле favorites.json появится информация о пользователе
